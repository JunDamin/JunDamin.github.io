Deploy

워래는 자동으로 deploy.yml이 생성 되는 듯 했는데 이유는 모르겠지만 내꺼에서는 안된다. 그렇지만 방법이 없는 건 아니다. 직접 .github/workflow에다가 만들어 넣으면 된다.

deploy.yml은 간단하게 살펴보면 일종의 스크립트다. 해야 할 일들을 적어 둘 수 있는데 몇가지 규칙이 있는 것 같다. 그 규칙을 몰라서 처음에는 난해해 보일 수 있는데 조금씩 뜯어서 보니 커널에 치는 것과 비슷해 보였다.
