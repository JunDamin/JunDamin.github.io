Chapter 7. How Sure Can We Be About What Is going On? Estimates and Intervals¶

불확실성을 인지하는 것이 중요하다¶

  • 통계조사 결과 평균이 3000명 감소
    • 에러가 $\pm 77,000$이라면? 3,000명이 감소했다고 말할 수 있는가?
  • 잘 실행된 조사인 경우, 우리는 샘플평균이 population mean에 가깝다고 추정한다
    • 참고사항: 샘플 파라미터는 로마자(m)로 population 파라미터는 그리스 문자($\mu$)로 쓰는 경우가 많다

Bootstrapping¶

  • 샘플(조사결과)가 하나 있을 때 어떻게 불확실성을 알 수 있을까?
    • 방법 1) 통계 모형을 설정해서 구한다.
      • 많은 가설을 도입해야 한다. 모집단의 분포가 어떨지 결정해야 한다.
    • 방법 2) 샘플을 다시 샘플링 한다면?
      • 적은 가설을 가지고 할 수 있다.
  • Bootstrapping: resampling from sample
    • 가정: 샘플분포가 모집단 분포와 비슷하다.
    • 방법
      • 샘플을 다시 샘플링해서 resample의 평균의 분포를 본다.
      • 그렇다면 모집단에서 sampling했을 때의 불확실성과 비슷할 것이다.
    • 참고
      • 중심극한정리를 확인할 수 있다.
      • 만약에 모집단이 극단적인 값을 가질 수 있는 분포라면 bootstraping은 조심해야 한다.

# conduct bootstrapping with sample size n

def bootstrap(population, size, n):
    output = []
    for i in range(n):
        output.append(np.random.choice(population, size=size).mean())
    return output

fig, ax = plt.subplots(4, 2, figsize=(16, 24))
# 1000 resamples
for i, size in enumerate([10, 100, 1000, 10000]):
    output = bootstrap(samples[i], size, 1000)
    ax[i][0].hist(output, bins=50, range=(0, 60))
    ax[i][0].set_xlim(0, 60)
    ax[i][0].set_title(f"1000 means of size = {size}")
    ax[i][0].vlines(population.mean(), 0, 1000, color="red")
# 100000 resamples
for i, size in enumerate([10, 100, 1000, 10000]):
    output = bootstrap(samples[i], size, 10000)
    ax[i][1].hist(output, bins=50, range=(0, 60))
    ax[i][1].set_xlim(0, 60)
    ax[i][1].set_title(f"10000 means of size = {size}")
    ax[i][1].vlines(population.mean(), 0, 10000, color="red")    
    
plt.subplots_adjust(wspace=0.3, hspace=0.5)
